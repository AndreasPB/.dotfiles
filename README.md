**.dotfiles and notes**

No capslock:
```bash
setxkbmap -option caps:escape
```

Vim Plug
```bash
curl -fLo ~/.vim/autoload/plug.vim --create-dirs \
    https://raw.githubusercontent.com/junegunn/vim-plug/master/plug.vim
```

Powerline font:
```bash
cd ~/Repo
git clone https://github.com/powerline/fonts
cd fonts/
./install.sh
```
